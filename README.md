# Asteroid Assault

Asteroid Assault is a fast-paced browser-based arcade shooter inspired by classic asteroid games. Built entirely with HTML5 Canvas and Vanilla JavaScript, the game challenges players to survive endless waves of asteroids while collecting power-ups and mastering physics-based ship control.

---

## Gameplay

You control a spaceship drifting through deep space. Destroy asteroids, avoid collisions, and survive as long as possible. Each cleared wave increases difficulty, testing your reflexes and control skills.

Asteroids break into smaller fragments when destroyed, making later stages more intense and chaotic.

---

## Features

- Smooth physics-based spaceship movement  
- Infinite asteroid waves with increasing difficulty  
- Real-time shooting and collision detection  
- Dynamic asteroid splitting system  
- Power-up system with temporary abilities:
  - Shield protection  
  - Rapid fire  
  - Spread shot  
- Adjustable thrust control  
- Score and lives tracking  
- Level progression system  
- Game over and restart functionality  
- Starfield animated background  
- Mobile touch controls and keyboard support  
- Responsive canvas resizing
